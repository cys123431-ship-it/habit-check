# Habit Check v0.2.1

## 변경 사항
- 앱 메인 화면/데이터 구조를 유지하면서 UX 문구 정리 및 가독성 개선
- 릴리즈용 앱 버전 업데이트 (`versionName: 0.2.1`, `versionCode: 2`)
- GitHub 릴리즈 절차 정비 및 릴리즈 노트 템플릿 적용

## 설치 안내
1. `app-release-signed.apk` 파일을 휴대폰으로 복사
2. 설정에서 출처를 알 수 없는 앱 설치 허용(최초 1회)
3. APK 설치 후 앱 실행

## 테스트 체크리스트
- 오늘 습관 추가/삭제 동작
- 오늘 체크 및 미완료 초기화 동작 확인
- 통계 화면의 7일 달성률/연속 기록 표시 확인
